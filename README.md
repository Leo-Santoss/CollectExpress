# CollectExpress - Aplicativo

O CollectExpress é um marketplace mobile desenvolvido para facilitar o aluguel de caçambas, conectando consumidores diretamente a caçambeiros da região de forma simples, rápida e segura. Ele permite navegar por caçambas, fazer pedidos, gerenciar endereços e avaliar os serviços prestados.

**Tecnologias Utilizadas:**
- **React Native & Expo:** Framework principal para construção da interface móvel e web.
- **Expo Router:** Roteamento baseado em arquivos (file-based routing) com rotas protegidas.
- **TypeScript:** Tipagem estática para maior confiabilidade e manutenção.
- **Axios & Context API:** Gerenciamento de estado (Carrinho, Auth) e comunicação HTTP com a API.
- **Expo Image & Vector Icons:** Ícones vetorizados e carregamento otimizado de imagens.
